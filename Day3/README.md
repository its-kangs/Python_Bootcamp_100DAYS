# Day 3: Treasure Island 🏝️

A CLI adventure game where your choices determine if you find the treasure or meet your end.

## Concepts Covered
- Conditional logic (`if` / `elif` / `else`)
- Nested `if` statements
- `.lower()` for input handling
- Multi-line strings & ASCII art

## How to Run
```bash
python treasure_island.py
```

## How to Win
There are 3 decision points — only one path leads to the treasure. Choose wisely.
